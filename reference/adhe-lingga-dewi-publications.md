## Publikasi Relevan IoT / Edge Computing (2024-2026)

### 1. Smart Air Monitoring with IoT-based MQ-2, MQ-7, MQ-8, and MQ-135 Sensors using NodeMCU ESP32

| Metadata | Detail |
|----------|--------|
| **Authors** | Easterline L.M., Putri A.A.-Z.R., Atmaja P.S., **Dewi A.L.**, Prasetyo A. |
| **Year** | 2024 |
| **Source** | *Procedia Computer Science*, 245, pp. 815-824 |
| **DOI** | [10.1016/j.procs.2024.10.308](https://doi.org/10.1016/j.procs.2024.10.308) |
| **Quartile** | Q2 (Conference Proceedings) |
| **Citations** | 22 |
| **Author Order** | 4 of 5 (Corresponding Author) |

**Abstract:** Implementasi IoT untuk sistem monitoring udara menggunakan NodeMCU ESP32 dengan sensor MQ-2, MQ-7, MQ-8, MQ-135, dan DHT11. Sistem mengumpulkan data sensor dan mentransmisikan ke platform cloud untuk analisis dan visualisasi. Hasil pengujian menunjukkan kadar CO2 di bawah 5,000 ppm, CO di bawah 25 ppm, dan gas lain dalam batas aman di Semarang. Error analisis suhu 1.05% dan kelembaban 1.04% terhadap data BMKG.

**Relevansi untuk RIIM:** ESP32 sebagai core controller, integrasi multi-sensor MQ (gas) via ADC, konektivitas WiFi ke cloud platform, arsitektur IoT end-to-end dari sensing hingga visualization.
